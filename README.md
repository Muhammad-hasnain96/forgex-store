# SilkLane — Pakistani Fashion

Online Pakistani clothing store converted from the original e-commerce template.

- `ironmart/` — Next.js 16 + SQLite full-stack store (main app)
- `ironmart-store.html` — legacy standalone demo
- `products.js` — product dataset for legacy demo

## Run the main app

```bash
cd ironmart
npm install
npm run db:reset
npm run dev
```

Open `http://localhost:3000`

For full setup, features, and deployment notes, see:

- `ironmart/README.md`
